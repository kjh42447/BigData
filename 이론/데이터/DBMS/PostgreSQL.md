# PostgreSQL

복잡한 쿼리와 대규모 DB를 다루기에 적합한 기능이 풍부한 오픈소스 객체 관계형 데이터베이스 관리 시스템(ORDBMS)이다. MySQL대비 많은 데이터 타입과 인덱스 등을 제공한다.

### PostgreSQL 장점
- ORDBMS : 테이블 상속, 함수 오버로딩 등을 지원
- 많은 데이터 형식 지원
- 많은 SQL기능 지원으로 복잡한 쿼리에 적합
- DB 사이즈에 제한을 두지 않기 때문에 대용량 DB에 적합
